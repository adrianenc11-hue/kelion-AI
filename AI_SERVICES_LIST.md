# K System - AI Services Used

## Primary AI Models

1. **OpenAI GPT-4o** - Main conversational AI
   - `chat.js` - Text chat
   - `realtime-token.js` - Real-time voice conversations
   - `vision.js` - Image analysis
   - `dalle.js` - Image generation
   - `whisper.js` - Speech-to-text

2. **Google Gemini**
   - `gemini-chat.js` - Text chat
   - `gemini-live-token.js` - Live streaming

3. **Anthropic Claude**
   - `claude-orchestrator.js` - Complex task orchestration
   - `claude-audit.js` - Code auditing

4. **Moonshot Kimi**

5. **DeepSeek**
   - `deepseek.js` - Code-focused AI

6. **ElevenLabs**
   - `elevenlabs-tts.js` - Text-to-speech
